model1.0

代码

kf.use('model1.0', function(){ var model = new kf.model(Obj, Params); model.show(data); });

Obj

Obj 展示模板内容的容器，值为object或id

Params

model 模板代码，此参数可选填，不存在时，默认获取 Obj 内textarea标签的值

data 模板数据，此参数可选填，存在时，自动触发 model.show(data)

callback 回调函数，此参数可选填，存在时，当模板显示完成后会回调此函数。

方法

show(data) 组装数据并显示
