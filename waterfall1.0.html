KF代码库

waterfall1.0

代码

kf.use('waterfall1.0', function(){ var waterfall = new kf.waterfall(Obj, Params); });

Obj

Obj 内容容器，值为object或id

Params

column 瀑布流列容器，此参数可选填，不存在时，先获取 .kf-waterfall-column 再获取 Obj 内li标签

model 模板代码，此参数可选填，不存在时，默认获取 Obj 内textarea标签的值

data 模板数据，此参数可选填，存在时，自动触发 waterfall.show(data)

callback 回调函数，此参数可选填，存在时，当模板显示完成后会回调此函数。

约定

model 模板中图片若需缓冲完成后再展示，则在img标签加上样式 loadimg

方法

show(data) 组装数据并显示
