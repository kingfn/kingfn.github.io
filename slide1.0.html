KF代码库

slide1.0

代码

kf.use('slide1.0', function(){ var slide = new kf.slide(Obj, Params); });

Obj

Obj 滑动模块内容的容器，值为object或id

约定

Obj 内元素约定：

.kf-slide-wrap 该元素为滚动元素容器，必选。

.kf-slide-content 该元素为滚动元素内容，可选，若不存在则获取 .kf-slide-wrap 元素下 ul 元素

.kf-slide-pannel 该元素为单个滚动元素内容，可选，若不存在则获取 .kf-slide-wrap 元素下 li 元素

Params

effect 切换效果，可选值：horizontal(横向滚动)、vertical(纵向滚动)，默认值 horizontal

autoPlay 是否自动播放，默认值 true

autoTime 自动切换时间差，单位毫秒，默认值 5000

autoRound 是否循环，默认值 true

iconState 是否展示小图标，默认值 true，为 true 时，脚本会在 .kf-slide-wrap 内自动创建类似以下元素

<ul class="kf-slide-icon"> <li data-index="0" class="current">1</li> <li data-index="1">2</li> <li data-index="2">3</li> </ul>

iconEvent 小图标触发事件，可选值：mouseover、click，默认值 mouseover

方法

prev 向前滑动一单位元素

next 向后滑动一单位元素

go(index) 滑动至指定序号元素
