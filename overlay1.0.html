KF代码库

overlay1.0

代码

kf.use('overlay1.0', function(){ var overlay = new kf.overlay(Params); });

Params

width 弹出框宽度，默认 auto

height 弹出框高度，默认 auto

close 弹出框关闭按钮，若不存在，默认值 .kf-overlay-close

mask 是否有蒙版，默认值 true

maskClose 蒙版是否可点击关闭弹出框，默认值 true

content 弹出框内容，若存在，则会触发 show(content)

方法

show(content) 显示内容，若 content 为空，则显示目前弹出框中的内容

hide 隐藏弹出框
