KF代码库

fade1.0

代码

kf.use('fade1.0', function(){ var fade = new kf.fade(Obj, Params); });

Obj

Obj 切换模块内容的容器，值为object或id

约定

Obj 内元素约定：

.kf-fade-wrap 该元素为切换元素容器，必选。

.kf-fade-content 该元素为切换元素内容，可选，若不存在则获取 .kf-fade-wrap 元素下 ul 元素

.kf-fade-pannel 该元素为单个切换元素内容，可选，若不存在则获取 .kf-fade-wrap 元素下 li 元素

Params

autoPlay 是否自动播放，默认值 true

autoTime 自动切换时间差，单位毫秒，默认值 5000

iconState 是否展示小图标，默认值 true，为 true 时，脚本会在 .kf-fade-wrap 内自动创建类似以下元素

<ul class="kf-fade-icon"> <li data-index="0" class="current">1</li> <li data-index="1">2</li> <li data-index="2">3</li> </ul>

iconEvent 小图标触发事件，可选值：mouseover、click，默认值 mouseover

方法

prev 向前切换一单位元素

next 向后切换一单位元素

go(index) 切换至指定序号元素
