kf.js

kf.js是KF代码库引入文件，引入此文件，便可以在网页中使用KF代码库里的所有组件。

需要注意的是，KF代码库组件全部寄存在全局变量kf下，故需要避免变量名kf被其他函数占用或重定义。

kf.js文件路径：

<script src="http://kf.fwowo.com/kf.js"></script>

属性

host 组件路径，默认值：http://kf.fwowo.com/，可通过修改此属性来更改组件调用路径，常用在调试本地组件。

debug 调试模式，默认false。true时引入的是组件源文件，false时引入的是组件min文件。

useArray 查看调用过的组件，只读。

方法

getStyle 加载引入css文件，加载成功回调callback函数。

use 引入组件，可单次引入多个组件，多个组件使用数组格式。

常用代码

<script src="http://kf.fwowo.com/kf.js"></script> <script> kf.getStyle('style.css', function(){ ... }); kf.use('module', function(){ var newModule = new kf.module(); ... }); </script>
