tab1.0

代码

kf.use('tab1.0', function(){ var tab = new kf.tab(Obj, Params); });

Obj

Obj 切换模块内容的容器，值为object或id

Params

eventType 触发动作，此参数可选填，不存在时，默认为 mouseover

index 初始显示序号，此参数可选填，不存在时，默认为 0

option 选择项对象，此参数可选填，不存在时，默认为 .kf-tab-option

optionClass 对象选中时样式名称，默认值 current

pannel 重置面板对象，此参数可选填，不存在时，默认为 .kf-tab-pannel

pannelClass 对象选中时样式名称，默认值 current

callback(index) 回调函数，此参数可选填，若存在，当 option 被触发，并执行结束之后会被调用执行。

方法

show(index) 触发指定序号的 option
